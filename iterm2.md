# iTerm 2

## 标签

```
Command + T : 新建标签
Command + W : 关闭标签
Command + 1 / 左右方向键 : 切换标签页
Command + Enter : 切换全屏
```

## 分屏

```
Command + D : 垂直分屏
Command + Shift + D : 水平分屏
Command + [ / ] : 切换屏幕
```

## 其他

```
Command + Shift + H : 查看最近命令
Ctrl + U : 清除当前行
Ctrl + R : 搜索命令历史
Ctrl + L : 清屏 1
Command + R ：清屏 2
```
